# Border



``` css
/* 속성 */
border-width: 1px; /* 두께 <length> |thin|medium|thick */
border-style:solid; /* 종류 none/ hidden/dotted/dashed/solid/double/groove/ridge */
border-color: #000; /* 색상 */

/* 축약형 */
/* border: border-width border-style border-color |initial|inherit */
border: 1px solid #000;
border-width: 6px; /* 보더 두께 */
border-width: 6px 3px; /* 보더 상하, 좌우 두께 */
border-width: 6px 3px 4px; /* 상, 좌우, 하 두께 */
border-width: 6px 3px 5px 1px; /* 상 하 좌 우 */
```
