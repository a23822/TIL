# itertools

### 1. product()

- ex

  ``` python
  product('ABCD', repeat=2)
  """
  AA AB AC AD BA BB BC BD CA CB CC CD DA DB DC DD
  """
  ```



###  2. permutations()

- ex

  ``` python
  permutations('ABCD', 2)
  """
  AB AC AD BA BC BD CA CB CD DA DB DC
  """
  ```



### 3. combinations()

- ex

  ``` python
  combinations('ABCD',2)
  """
  AB AC AD BC BD CD
  """
  ```



### 4.combinations_with_replacement()

- ex

  ``` python
  combinations_with_replacement('ABCD', 2)
  """
  AA AB AC AD BB BC BD CC CD DD
  """
  ```
